Data Form

Data forms allow you to add structured data to topics

The data stored in the form fields can be used to search and filter topics.
The combination of structured data and search queries are the base for building database applications.

Read more about Data Forms

Attachments

You can store Attachments in any format

Attachments are stored under revision control: uploads are automatically backed up; all previous versions of a modified file can be retrieved.

Read more about attachments

Permissions

You can define users or groups who are allowed to view or edit this topic

Permissions are based on the familiar concept of users and groups. Users are defined by their WikiNames. They can then be organized in unlimited combinations by inclusion in one or more user Groups. For convenience, Groups can also be included in other Groups. But remember: Open, freeform editing is the essence of WikiCulture.

Read more about Permissions

Topic Settings

Topic settings allow you to modify display options for normal page view and edit mode

Topic settings allow you to show or hide different parts of a topic (like breadcrumbs or the sidebar), or apply a special ViewTemplate or EditTemplate. You can also specify which editor you want to use for modifiying this topic.

Read more about topic settings
